# Sistema-de-Inventario
Pagina Web,  nuestra idea es implementar un sistema de tabletas, las cuales tendran acceso directo a una pagina web, donde podran ver en tiempo real, la cantidad de sus productos y su estado, y donde podran realizar el inventario de manera mas eficiente.

Sistema web de inventarios desarrollado con FastAPI, PostgreSQL, ActiveMQ y frontend HTML/CSS/JS.

## Funciones principales

- Login de usuarios con contraseña cifrada usando Argon2.
- Roles: Administrador, Operador y Auditor.
- Alta de productos.
- Registro de entradas de inventario.
- Consulta de movimientos.
- Registro de inventario físico.
- Consulta de inventarios realizados.
- Gestión de usuarios.
- Auditoría mediante logs y triggers en PostgreSQL.

## Tecnologías usadas

- FastAPI
- PostgreSQL
- ActiveMQ
- HTML, CSS y JavaScript
- Pydantic
- Argon2
